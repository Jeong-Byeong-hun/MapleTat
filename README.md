# MapleTat 
자신의 메이플 캐릭터를 검색해보자 !


## 개발기간
* 2023.11 ~ 2024.01


### 개발환경
- Android Studio - Android Studio Giraffe | 2022.3.1 Patch 1
- AGP Version - 8.1.1
- Gradle Version - 8.3
- CompileSdk - 34
- minSdk - 28
- targetSdk - 33
- JDK - java8

**Tech Stack:**

- Kotlin
- Jetpack Compose
- Glide
- Hilt
- Coroutine 
- Retrofit2
- Jetpack Compose Navigation
- Room
- MVVM
